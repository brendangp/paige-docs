Paige is a WhatsApp bot builder platform designed for teams that want to automate conversations on WhatsApp without writing infrastructure from scratch. You connect your WhatsApp Business number, describe what you want your bot to do, and Paige handles the rest — from generating the bot code to storing conversation data to delivering messages at scale.

## What you can build

With Paige you can automate customer support, run lead capture flows, send appointment reminders, collect feedback, and build any other conversational experience that fits your business. Whether you need a simple FAQ bot or a multi-step onboarding flow with conditional logic, Paige gives you the tools to build it.

## Key features

**AI bot builder** — Chat with the AI assistant to write and edit your bot's code in plain language. You describe the behavior you want, and the AI translates it into working bot logic. You can refine it at any time by continuing the conversation.

**Visual flow builder** — Create multi-screen interactive forms that run natively inside WhatsApp using WhatsApp Flows. Flows let users tap through structured screens — picking options, submitting details, confirming bookings — without leaving the chat.

**Built-in database** — Every project includes its own database. You can create tables, store contact data, log orders, or persist any state your bot needs, all from within the Paige dashboard.

**Conversation and contact management** — View every conversation your bot has had, browse your contact list, and inspect message history from a single dashboard.

**WhatsApp Business API integration** — Paige connects to Meta's WhatsApp Business API through an embedded signup flow. You link your existing WhatsApp Business Account or create a new one, and Paige manages the webhook connection and token handling for you.

**Message templates** — Create and manage Meta-approved message templates for sending outbound notifications to users who haven't messaged you recently.

**Secrets manager** — Store API keys and other sensitive values securely. Your bot code can reference secrets by name without exposing the underlying values.

**Execution logs and version history** — Every time your bot runs, Paige logs the execution. If something breaks, you can inspect the logs and roll back to any previous version of your bot code with one click.

**Live testing with Paige Dev** — Link your personal phone number to test your bot in a real WhatsApp conversation before deploying it to users.

**Team workspaces** — Invite teammates to a shared workspace so multiple people can collaborate on the same projects.

## Who Paige is for

Paige is built for product and operations teams, freelancers, and agencies that need to ship WhatsApp automations quickly. You don't need to manage webhooks, write a backend service, or set up a database — Paige provisions all of that when you create a project.
